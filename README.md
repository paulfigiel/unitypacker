# unitypacker

Create .unitypackages from command line

```diff
USAGE:
    unitypacker-macos [OPTIONS] <SUBCOMMAND>

OPTIONS:
    -h, --help                                       Print help information
    -u, --unity-project-root <UNITY_PROJECT_ROOT>    Root of the unity project
    -V, --version                                    Print version information

SUBCOMMANDS:
    help    Print this message or the help of the given subcommand(s)
    pack    Create a .unitypackage
```
